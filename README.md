## <a name="program" /> Программа курса 
Тема 1.
Краткая история нейросетевого подхода в компьютерном зрении.

Тема 2.
Постановка основных задач компьютерного зрения.

Тема 3.
Сверточные нейронные сети. Типы слоев. 

Тема 4.
Стохастический градиентный спуск. Оптимизационные алгоритмы.

Тема 5.
Введение в нейросетевые фреймворки на примере Keras.

Тема 6.
Метод обратного распространения ошибки.

Тема 7.
Способы инициализации весов нейросетей.

Тема 8.
Классификация изображений.

Тема 9.
Детектирование объектов на изображениях. MAP, однопроходные, двухпроходные алгоритмы.

Тема 10.
Распознавание лиц.

Тема 11.
Сегментация объектов на изображениях.

Тема 12.
Генеративные состязательные сети.

Тема 13.
Автоэнкодер. Вариационный автоэнкодер.

Тема 14.
Улучшение качества изображений. Сверхразрешение, деблюр, денойз.

Тема 15.
Состязательные атаки.

Тема 16.
Сжатие и ускорение нейросетей.
